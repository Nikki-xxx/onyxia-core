# USUARIO

Versión: 1.0

Estado: En desarrollo

---

# Objetivo

Un Usuario representa cualquier persona registrada dentro de la plataforma Onyxia.

Todo usuario posee una identidad única (Onyxia ID) y uno o más roles.

La identidad nunca cambia.

Los roles pueden cambiar durante la vida del usuario.

---

# Información básica

Todo usuario tendrá como mínimo:

- Onyxia ID
- Telegram ID
- Nombre visible
- Username (opcional)
- País
- Idioma
- Fecha de registro
- Estado
- Roles
